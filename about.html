About Erin's CookBook!

To create a recipe:

First go to the Select Ingredients page. From here you can check if all the ingredients are available in the recipe book. If an ingredient is not listed, insert the ingredient first (see below). If all ingredients are available, select each ingredient used in your recipe and then click submit. This will direct you to a page where you can enter the amount of the ingredient required for the recipe. Once you have done this, you will submit the recipe, and viola, the recipe is now stored.

To insert an ingredient:

You will need to insert the ingredient before you can use it in a recipe. To do this, go to the Insert Ingredient page. From here you can elect to specify a cost and yield of the ingredient. In this cook book, costing information is provided based on current costing information the the culinary department of Collin County Community College uses to do costing converions on the recipes prepared in class.

When inserting an ingredient, make sure the appropriate category is selected to make it easier to locate the ingredient when inserting a recipe. This will make it easier for future projects on this site to create a Mise En Place for our cooks using this site. As you can see, someone has already entered butter as vegetable. Although most users are located in Texas, butter is still not considered a vegetable.

To view all recipes:

The current list of recipes in the cook book are listed in the View All Recipes page. This page sorts the recipe alphabetically. If you would like to view only a particular recipe, go to the View Selected Recipe page. From here, you can select a recipe and choose a few different options to customize your selection.

To delete:

To delete a recipe go to the Delete Recipe page. This page will allow you to select a specific recipe to delete. To delete a specific ingredient, go to the Delete Ingredients page. We should warn you, that if any recipe contains the ingredient you are trying to delete, you will not be able to delete that ingredient.
